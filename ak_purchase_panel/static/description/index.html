Purchase Panels

Create purchase panels/committees. Invite panel members to review the bid evaluation process.


This module extends the Bid Evaluation app by enabling purchase panels to participate in evaluating bids/quotations for a given tender/purchase agreement. The app allows creation of panels or comittees, and a validation workflow for approving each evaluation. The app also extends bid evaluation reports by adding purchase panel details into them.

The app has been tested on both the Community and Enterprise versions.



How to Use the App

Purchase Panels

Once this module is installed on top of the bid evaluation app, you can create panels from the 'Purchase Panel' menu under the configuration tab. Having specified a panel on an agreement, each evaluation would need to be reviewed by all panel members before being fully validated. Panel member can be assigned the user group 'Purchase Panel Member' which will grant them read access to agreement and quotations that they have been invited to as well as their evaluation records. Members are notified through an activity created on each evaluation.
